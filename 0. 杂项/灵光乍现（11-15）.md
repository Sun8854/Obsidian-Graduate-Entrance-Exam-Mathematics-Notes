## 积分不等式证明方法
### 解题方法选择

| 题目特征                                | 选用的方法        | 核心操作                                     |
| ----------------------------------- | ------------ | ---------------------------------------- |
| 连续函数 + 不等式关系<br/>证积分值与常数            | **单调性法**     | 令 $F(x)=\int_a^x f(t)dt$，求导研究单调性         |
| 一阶可导 + 某端点 $f=0$<br/>估值积分与导数界       | **拉格朗日中值定理** | 分段用 $f(x)-f(x_0)=f'(\xi)(x-x_0)$，取绝对值放缩  |
| 二阶及以上可导 + 某点函数值简单<br/>原函数难积分        | **泰勒公式**     | 展开成多项式，积分后对余项放缩                          |
| 被积函数为 $f(x)$·三角函数                   | **分部积分**     | 选 $u=f(x)$，$dv=\sin(nx)dx$，把 $f$ 转成 $f'$ |
| $f(a)=f(b)=0$<br/>沟通 $f$ 与 $f'$ 的积分 | **牛顿莱布尼兹公式** | $f(x)=\int_a^x f'(t)dt$，两边估计后相加消去中间量     |

**老王教你记**：看到"连续 + 变量化"就单调性；"一阶导 + 端点为0"就拉格朗日；"二阶导 + 展开多项式"就泰勒；"f·三角"就分部积分；"两端为0"就牛顿莱布尼兹。拿到题先扫特征，对号入座。
