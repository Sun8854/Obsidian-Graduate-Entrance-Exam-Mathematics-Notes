## 积分不等式证明方法总结

这门课的核心：**定积分是个数**，所以不等式证明的套路就是把积分转成函数，研究函数性态。

---

### 方法一：单调性法（最常用）

**套路**：把积分上限变量化 → 令 $F(x) = \int_a^x f(t)dt$ → 求导研究单调性

**典型特征**：
- 题目给的是连续函数 + 一个不等式关系
- 要证 $\int_a^b f(x)dx$ 和某个常数的关系

**例 11.7 核心步骤**：
1. 第 1 问：积分保号性，直接得 $0 \leq \int_a^x g(t)dt \leq x-a$
2. 第 2 问：设 $F(x) = \int_a^{a+\int_a^x g(t)dt} f(u)du$，求导
3. 关键发现：$F'(x) = [a + \int_a^x g(t)dt - x] \cdot g(x)$，而括号里用第 1 问的结论 ≤ 0
4. $g(x) \geq 0$，所以 $F'(x) \leq 0$ → $F(x)$ 单调递减
5. $F(a) = 0$，$F(b) \leq 0$ 正好是要证的结果

**避坑**：上限带积分的复合上限求导要会 — $F'(x) = f(\text{上限}) \cdot \text{上限对x求导}$

---

### 方法二：拉格朗日中值定理

**套路**：$f(x) - f(x_0) = f'(\xi)(x-x_0)$，用于沟通 $f$ 和 $f'$

**典型特征**：
- 条件给一阶可导 + 某端点函数值简单（常为 0）
- 需要估值 $\int_a^b f(x)dx$ 与导数界的关系

**例 11.8 核心步骤**（$f(0)=f(1)=0$，$|f'(x)| \leq M$）：
1. 分段：在 $[0,x]$ 和 $[x,1]$ 上分别用拉格朗日
   - $f(x) = f'(\xi_1) \cdot x$
   - $f(x) = f'(\xi_2) \cdot (x-1)$
2. 取绝对值：$|f(x)| \leq Mx$ 和 $|f(x)| \leq M(1-x)$
3. 积分拆分：$|\int_0^1 f(x)dx| \leq \int_0^x Mxdx + \int_x^1 M(1-x)dx = M[x^2/2 + 1/2 - x^2/2]$
4. 关键估值：$[x^2/2 + (1-x)^2/2] \geq 1/4$（因为这是个关于 $x=1/2$ 的二次函数，最小值为 $1/4$）
5. 故 $|\int_0^1 f(x)dx| \leq M/4$

**避坑**：分段点 $x$ 是任取的，最后取最小值（$1/4$）做上界估计

---

### 方法三：泰勒公式

**套路**：把难积分的函数展开成多项式，积分多项式容易做

**典型特征**：
- 二阶以上可导 + 某点函数值简单（常为 0）
- 配合放缩法估值

**例 11.9 核心步骤**（$f(1)=0$，$|f''(x)| \leq M$ 在 $[0,2]$ 上）：
1. 在 $x_0=1$ 处展开：$f(x) = f'(1)(x-1) + \frac{1}{2}f''(\xi)(x-1)^2$
2. 积分：$\int_0^2 f(x)dx = f'(1)\int_0^2 (x-1)dx + \frac{1}{2}\int_0^2 f''(\xi)(x-1)^2 dx$
3. 第一项：$\int_0^2 (x-1)dx = 0$（**对称性**，奇函数在对称区间积分为 0，几何意义要会看）
4. 第二项：$|\int_0^2 f''(\xi)(x-1)^2 dx| \leq \frac{1}{2}M\int_0^2 (x-1)^2 dx = \frac{1}{2}M \cdot \frac{2}{3} = \frac{M}{3}$

**避坑**：$f''(\xi)$ **不能**提到积分号外面，因为 $\xi$ 是 $x$ 的函数，随 $x$ 变化而变化

---

### 方法四：积分法（分部积分）

**套路**：被积函数是 $f(x) \cdot \text{三角函数}$，用分部积分把 $f(x)$ 转成 $f'(x)$

**例 11.10 核心步骤**（$f'(x) \geq 0$，证 $|\int_0^{2\pi} f(x)\sin(nx)dx| \leq \frac{2}{n}[f(2\pi)-f(0)]$）：
1. 分部积分：$\int f(x)\sin(nx)dx = -\frac{1}{n}f(x)\cos(nx) + \frac{1}{n}\int f'(x)\cos(nx)dx$
2. 边界项：$-\frac{1}{n}[f(2\pi)-f(0)]$（因为 $\cos(2n\pi)=1$，$\cos(0)=1$）
3. 加绝对值，用 $|\int| \leq \int |\cdot|$ 和 $f'(x) \geq 0$ 放缩
4. 结果：$\frac{1}{n}[f(2\pi)-f(0)] + \frac{1}{n}\int_0^{2\pi} f'(x)dx = \frac{2}{n}[f(2\pi)-f(0)]$

---

### 方法五：牛顿莱布尼兹公式

**套路**：$f(x) = f(a) + \int_a^x f'(t)dt$，直接沟通 $f$ 和 $f'$

**典型特征**：
- 条件给 $f(a) = f(b) = 0$
- 要证 $|f(x)| \leq \frac{1}{2}\int_a^b |f'(t)|dt$

**核心步骤**：
1. $f(x) = \int_a^x f'(t)dt$（因为 $f(a)=0$）
2. $|f(x)| \leq \int_a^x |f'(t)|dt$
3. 同理 $|f(x)| = |f(b)-f(x)| \leq \int_x^b |f'(t)|dt$
4. 两式相加：$2|f(x)| \leq \int_a^b |f'(t)|dt$

### 方法六：零点定理 + 单调性（变限积分方程根的个数）

**典型特征**：方程由变限积分构成，研究根的个数（选填题）。

**套路**：
1. 设 $F(x) = $ 方程左端，变量化
2. 端点异号 → 零点定理 → 至少一个根
3. 求导证单调 → 唯一性

**题目 1 核心**：
- $F(0) = \int_a^0 \frac{1}{\sqrt{4a^2-t^2}}dt < 0$（调换上下限变负号）
- $F(a) = \int_0^a \sqrt{4a^2-t^2}dt > 0$
- $F'(x) = \sqrt{4a^2-x^2} + \frac{1}{\sqrt{4a^2-x^2}} > 0$，单调递增 → 唯一根

---

### 方法七：对称区间积分符号判定

**核心公式**（必须背！）：
$$\int_{-a}^{a} \varphi(x)dx = \int_{0}^{a} [\varphi(x) + \varphi(-x)]dx$$

**关键性质**：$\sin x$ 在 $(0,\pi)$ 恒正；$\cos x$ 在 $[0,\pi]$ 有正有负。

**题目 2 核心**：
- $I_1$：用公式 + $\sin x>0$ + $f'(x)>0$（即 $f$ 单增）$\Longrightarrow I_1>0$
- $I_2$：$\cos x$ 不能直接用，先凑微分 $\cos x dx = d(\sin x)$，分部积分把 $f$ 转成 $f'$，再用公式 + $f''(x)<0$（即 $f'$ 单减）$\Longrightarrow I_2>0$

**老王提醒**：见到对称区间不要只想着奇偶性，这个公式是通法。sin 恒正是宝贝，cos 变号所以要转成 sin。

---

### 方法八：周期函数判定（求导法）

**核心结论**：$f(x)$ 以 $T$ 为周期 $\Longleftrightarrow$ 对任意 $a$，$\int_a^{a+T} f(x)dx$ 为常数。

**题目 3 充分性证明关键**：
- 记 $\int_a^{a+T} f(x)dx = C$
- 对 $a$ 求导：$f(a+T) - f(a) = 0$（左边求导用变限积分求导公式，右边常数求导为 0）
- 对任意 $a$ 成立，故 $f$ 以 $T$ 为周期

**老王提醒**：把常数 $a$ 当变量，对等式两端同时求导，是此类证明的核心操作。

---

### 方法九：变限积分的积分（分部积分 + 常数可进出积分号）

**题目 4 核心**：
- **方法一**：求导比较法——两边对 $x$ 求导，$\varphi'(x) = \psi'(x)$ 且初始值均为 0
- **方法二**：直接分部积分——把里层变限积分当 $u$，外表分积分当 $\int v du$
  $$\int_0^x \left[\int_0^t f(u)du\right]dt = \left[t\cdot\int_0^t f(u)du\right]_0^x - \int_0^x t\cdot f(t)dt = \int_0^x f(t)(x-t)dt$$

**老王提醒**：$x$ 在积分号外面时可以拿进去当常数（对积分变量而言），这是处理此类问题的常用变形。

---

### 方法十：中值等式——罗尔定理的辅助函数构造

**题目 5 核心**：
1. **改字母**：把 $\xi$ 换成 $x$，移项使一边为 0
2. **凑导数**：观察 $f(x)\int_x^b g(t)dt$ 是谁的导数？——变下限求导多一个负号
   $$[F(x)]' = f(x)\int_x^b g(t)dt - g(x)\int_a^x f(t)dt$$
   故 $F(x) = \int_a^x f(t)dt \cdot \int_x^b g(t)dt$
3. **验证**：$F(a)=F(b)=0$（上下限相等），罗尔定理得证

**老王提醒**：构造辅助函数的核心是"凑导数公式"。变上限 $\int_a^x f(t)dt$ 求导得 $f(x)$；变下限 $\int_x^b g(t)dt$ 求导得 $-g(x)$。看到导数公式要敏感。

---

### 方法十一：倒代换 + 积分与字母无关（反常积分计算）

**题目 6 核心**：
1. **相等**：$x=1/t$ 倒代换，$\int_0^{+\infty} \frac{dx}{1+x^4} = \int_0^{+\infty} \frac{t^2}{1+t^4}dt$；积分与字母无关，换回 $x$ 即可
2. **计算**：两个不好算的相加除 2：
   $$\int_0^{+\infty} \frac{dx}{1+x^4} = \frac{1}{2}\int_0^{+\infty} \frac{1+x^2}{1+x^4}dx$$
   同除 $x^2$ 凑微分：$\frac{d(x-1/x)}{(x-1/x)^2+2}$，令 $u=x-1/x$，化为 $\int_{-\infty}^{+\infty} \frac{du}{u^2+2}$

**老王提醒**：倒代换 $x=1/t$ 专门对付分母高次方。经典积分 $\int_0^{+\infty} \frac{dx}{1+x^4} = \frac{\sqrt{2}\pi}{4}$ 要背。
