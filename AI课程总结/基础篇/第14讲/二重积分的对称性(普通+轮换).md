## 一、普通对称性

> 教材原话 (full.md:21-34):设区域 $D$ 关于 $y$ 轴对称,取对称的两块小面积 $\mathrm{d}\sigma$,对称点分别为 $(x,y)$ 与 $(-x,y)$,则对称点处的高分别为 $f(x,y)$ 与 $f(-x,y)$。

### 1. 物理类比(老王"大面包 + 小薯条")

把二重积分想成一个**大面包(曲顶柱体)**,每一块 $\mathrm{d}\sigma$ 是一条**小薯条(小竖条)**。

- 普通对称性的 $D$ 是关于某条线/某个点**对称**的
- 对称位置的两条小薯条,它们的**底面积** $\mathrm{d}\sigma$ 一样
- 但是**高** $f$ 在两个对称点可能相同,也可能相反

**直观图示**(曲顶柱体由 $\mathrm{d}\sigma$ 累加而成):

![Fig 1: 二重积分 = 小薯条累加成大面包](assets/fig1_volume.png)

### 2. 关于 $y$ 轴(即 $x=0$)对称
![](assets/二重积分的对称性(普通+轮换)/file-20260707142531151.png)

$$\boxed{\iint_{D} f(x,y)\,\mathrm{d}\sigma = \begin{cases} 2 \displaystyle\iint_{D_1} f(x,y)\,\mathrm{d}\sigma, & f(x,y) = f(-x,y) \\[1ex] 0, & f(x,y) = -f(-x,y) \end{cases}}$$

其中 $D_1$ 是 $D$ 在 $y$ 轴**右侧**的部分(关于 $y$ 轴对称点 $(x,y)$ 与 $(-x,y)$)。教材图 14-4 (full.md:25)

> [!tip] 口诀
> **偶倍奇零** — 代入对称点后函数值**相等**就**2 倍**,**相反**就**0**。

> [!warning] 老王原话 ⚠️
> 严格说,**多元函数没有"奇函数/偶函数"这个概念**。这里只是借 1 元定积分的术语方便记忆,口诀性质,不能当作定义。

### 3. 对称轴的平移推广

#### 3.1 关于 $x = a\ (a \neq 0)$ 对称

设 $D$ 关于 $x = a$ 对称,对称点为 $(x,y)$ 与 $(2a-x,y)$(把 $x=0$ **平移** $a$ 个单位即得)。教材 (full.md:52-58)

$$\iint_{D} f(x,y)\,\mathrm{d}\sigma = \begin{cases} 2 \displaystyle\iint_{D_1} f(x,y)\,\mathrm{d}\sigma, & f(x,y) = f(2a-x,y) \\[1ex] 0, & f(x,y) = -f(2a-x,y) \end{cases}$$

其中 $D_1$ 是 $D$ 在 $x=a$ **右侧**的部分。

> [!important] 对称点公式(必背)
> 已知点 $(x,y)$,关于 $x=a$ 的对称点是 $\boxed{(2a-x,\ y)}$
> 几何意义:两点的 $x$ 坐标以 $x=a$ 为中点,即 $a = \dfrac{x+(2a-x)}{2}$。

#### 3.2 关于 $y = a\ (a \neq 0)$ 对称

设 $D$ 关于 $y = a$ 对称,对称点为 $(x,y)$ 与 $(x,2a-y)$。教材 (full.md:70-78)

$$\iint_{D} f(x,y)\,\mathrm{d}\sigma = \begin{cases} 2 \displaystyle\iint_{D_1} f(x,y)\,\mathrm{d}\sigma, & f(x,y) = f(x,2a-y) \\[1ex] 0, & f(x,y) = -f(x,2a-y) \end{cases}$$

其中 $D_1$ 是 $D$ 在 $y=a$ **上侧**的部分。

### 4. 关于 $x$ 轴(即 $y=0$)对称

$$\iint_{D} f(x,y)\,\mathrm{d}\sigma = \begin{cases} 2 \displaystyle\iint_{D_1} f(x,y)\,\mathrm{d}\sigma, & f(x,y) = f(x,-y) \\[1ex] 0, & f(x,y) = -f(x,-y) \end{cases}$$

其中 $D_1$ 是 $D$ 在 $x$ 轴**上侧**的部分。教材图 14-? (full.md:66)

### 5. 关于原点对称

对称点 $(x,y)$ 与 $(-x,-y)$。教材 (full.md:86-90)

$$\iint_{D} f(x,y)\,\mathrm{d}\sigma = \begin{cases} 2 \displaystyle\iint_{D_1} f(x,y)\,\mathrm{d}\sigma, & f(x,y) = f(-x,-y) \\[1ex] 0, & f(x,y) = -f(-x,-y) \end{cases}$$

### 6. 关于 $y=x$ 对称

对称点 $(x,y)$ 与 $(y,x)$(交换坐标)。教材 (full.md:94-106)

$$\iint_{D} f(x,y)\,\mathrm{d}\sigma = \begin{cases} 2 \displaystyle\iint_{D_1} f(x,y)\,\mathrm{d}\sigma, & f(x,y) = f(y,x) \\[1ex] 0, & f(x,y) = -f(y,x) \end{cases}$$

> [!warning] **关于 $y=x$ 对称的本质条件**
> 把描述 $D$ 的方程中的 $x,y$ 对调后,表达式**不变**。这样才能说"区域 $D$ 关于 $y=x$ 对称"。(老王强调"程序化判断")

### 7. ⚠️ 普通对称性使用的"两步程序"(老王总结)

> 教材原话 (full.md:80-84):"关键两点:①看 $D$ 关于谁对称;②将点代入 $f(x,y)$,相等则 2 倍,相反则为 0。"

| Step | 动作 | 判定 |
|------|------|------|
| **Step 1** | 看 $D$ 关于**谁**对称 | 是 $y$ 轴 / $x$ 轴 / 原点 / $y=x$ / $x=a$ / $y=a$? |
| **Step 2** | 把**对称点**代入 $f(x,y)$ | 代入值**相等** → $\iint_D f = 2\iint_{D_1} f$;**相反** → $\iint_D f = 0$ |

> [!tip] 老王原话
> "今后解题,这就是一个程序问题——你告诉计算机(或者你自己)怎么操作,而不是凭感觉乱猜。"

![Fig 2: 6 种常见对称性速查](assets/fig2_six_symmetries.png)

### 8. 普通对称性速查表

| 对称类型 | 对称点 $(x,y) \mapsto$ | $f$ 代入条件 | 结论 |
|---------|------------------------|-------------|------|
| 关于 $y$ 轴 | $(-x, y)$ | $f(-x,y) = \pm f(x,y)$ | $=2\iint_{D_1} f$ 或 $=0$ |
| 关于 $x=a$ | $(2a-x, y)$ | $f(2a-x,y) = \pm f(x,y)$ | $=2\iint_{D_1} f$ 或 $=0$ |
| 关于 $x$ 轴 | $(x, -y)$ | $f(x,-y) = \pm f(x,y)$ | $=2\iint_{D_1} f$ 或 $=0$ |
| 关于 $y=a$ | $(x, 2a-y)$ | $f(x,2a-y) = \pm f(x,y)$ | $=2\iint_{D_1} f$ 或 $=0$ |
| 关于原点 | $(-x, -y)$ | $f(-x,-y) = \pm f(x,y)$ | $=2\iint_{D_1} f$ 或 $=0$ |
| 关于 $y=x$ | $(y, x)$ | $f(y,x) = \pm f(x,y)$ | $=2\iint_{D_1} f$ 或 $=0$ |

## 二、例 14.3:同一函数在三个区域上的积分比大小

> 教材原题 (full.md:110-112)

**Geometry of the 3 regions** (core: see the $y=x$ diagonal and the $y=\sqrt{x}$ helper curve):

![Fig 3: 例 14.3 三个区域](assets/fig3_three_regions.png)

**对照总结**(必背):

| 区域 | 形状 | 关于 $y=x$ 对称? | $f(x,y)=\sqrt[3]{x-y}$ 在剩余部分的符号 |
|------|------|---------------|-----------------------------------|
| $D_1$ | 1×1 正方形 | ✓ 整体对称 | $J_1 = 0$ |
| $D_2$ | $\sqrt{x}$ 下方 | ✗ 切 $y=x^2$ 后 $D_{21}$ 对称, $D_{22}$ 不对称 | $D_{22}$ 上 $x>y \Rightarrow f>0 \Rightarrow J_2 > 0$ |
| $D_3$ | $\sqrt{x}$ 上方至 1 | ✗ 切 $y=\sqrt{x}$ 后 $D_{32}$ 对称, $D_{31}$ 不对称 | $D_{31}$ 上 $x<y \Rightarrow f<0 \Rightarrow J_3 < 0$ |

### 1. 题目 ✅ 教材核对 (full.md:110)

设
$$J_i = \iint_{D_i} \sqrt[3]{x - y}\,\mathrm{d}x\mathrm{d}y \quad (i=1,2,3)$$

其中
$$\begin{cases} D_1 = \{(x,y) \mid 0 \leqslant x \leqslant 1,\ 0 \leqslant y \leqslant 1\} \\[0.5ex] D_2 = \{(x,y) \mid 0 \leqslant x \leqslant 1,\ 0 \leqslant y \leqslant \sqrt{x}\} \\[0.5ex] D_3 = \{(x,y) \mid 0 \leqslant x \leqslant 1,\ x^2 \leqslant y \leqslant 1\} \end{cases}$$

求 $J_1, J_2, J_3$ 的大小关系:
(A) $J_1 < J_2 < J_3$ (B) $J_3 < J_1 < J_2$ (C) $J_2 < J_3 < J_1$ (D) $J_2 < J_1 < J_3$

### 2. 老王的"程序"思路

> **老王总结(必背)**:比大小的程序—— 第一步看 $D$ 对称不; 第二步把对称点代入 $f$,看相同还是相反。

### 3. Step 1:判断 $J_1$

#### 3.1 三个区域的几何形状

| 区域 | 形状 | 边界 |
|------|------|------|
| $D_1$ | 单位正方形 $[0,1]\times[0,1]$ | 4 条直线 |
| $D_2$ | 由 $y=\sqrt{x}$ 与 $x$ 轴围成 | $x\in[0,1]$, $y\in[0,\sqrt{x}]$ |
| $D_3$ | 由 $y=x^2$ 与 $y=1$ 围成 | $x\in[0,1]$, $y\in[x^2,1]$ |

#### 3.2 判定 $D_1$ 关于 $y=x$ 对称

**判断方法**:把描述 $D_1$ 的不等式 $\{0 \leqslant x \leqslant 1,\ 0 \leqslant y \leqslant 1\}$ 中的 $x,y$ **互换**位置,得到 $\{0 \leqslant y \leqslant 1,\ 0 \leqslant x \leqslant 1\}$,**完全相同**。所以 $D_1$ 关于 $y=x$ 对称。

#### 3.3 把对称点代入 $f$

- 代入 $(x,y)$: $f(x,y) = \sqrt[3]{x-y}$
- 代入对称点 $(y,x)$: $f(y,x) = \sqrt[3]{y-x} = -\sqrt[3]{x-y}$

两者**相反**!

$$\therefore J_1 = 0$$

### 4. Step 2:判断 $J_2$

#### 4.1 $D_2$ 自身不对称 —— 创造条件

老王原话:"虽然 $D_2$ 自己不对称,但是你可以作**辅助曲线 $y = x^2$**(注意 $y=\sqrt{x}$ 和 $y=x^2$ 恰好**互为反函数**,关于 $y=x$ 对称)。"

#### 4.2 作辅助线 $y=x^2$ 把 $D_2$ 切成两块

| 子区域 | 范围 | 关于 $y=x$ 的对称点位置 |
|--------|------|----------------------|
| $D_{21}$ | $y$ 在 $[\sqrt{x}, x^2]$ 段 | 这部分**关于 $y=x$ 对称**(对应 $D_3$ 中的 $D_{31}$) |
| $D_{22}$ | $y$ 在 $[0, \sqrt{x}]$ 段 | 这部分**单独构成一块** |

#### 4.3 累加

$$J_2 = \iint_{D_{21}} \sqrt[3]{x-y}\,\mathrm{d}\sigma + \iint_{D_{22}} \sqrt[3]{x-y}\,\mathrm{d}\sigma$$

由普通对称性:**$D_{21}$ 上的积分 = 0**(因为这部分关于 $y=x$ 对称,且 $f$ 满足 $f(y,x) = -f(x,y)$)。

#### 4.4 判定 $D_{22}$ 上 $f$ 的符号

$D_{22}$ 是 $0 \leqslant y \leqslant \sqrt{x},\ 0 \leqslant x \leqslant 1$ 的区域,位于 $y=\sqrt{x}$ 的**下侧** —— 即 $y \leqslant \sqrt{x} < x$ 这一侧(因为 $\sqrt{x} < x$ 当 $x > 0$ 在 $[0,1]$ 内)。

> [!tip] 关键观察 ⚠️
> 在 $D_{22}$ 上,$\sqrt{x} < x$ 当 $x > 0$ 时严格成立 → $x > y$ → $x - y > 0$ → $\sqrt[3]{x-y} > 0$。

由保号性:
$$J_2 = \iint_{D_{22}} \sqrt[3]{x-y}\,\mathrm{d}\sigma > 0$$

> 教材原话 (full.md:122-127):由保号性知,$J_2 = \iint_{D_{22}} \sqrt[3]{x-y}\,\mathrm{d}x\mathrm{d}y > 0$。

### 5. Step 3:判断 $J_3$

#### 5.1 作辅助线 $y=\sqrt{x}$ 把 $D_3$ 切成两块

| 子区域 | 范围 | 说明 |
|--------|------|------|
| $D_{31}$ | $y$ 在 $[\sqrt{x}, 1]$ 段 | 位于 $y=\sqrt{x}$ 的**上侧** |
| $D_{32}$ | $y$ 在 $[x^2, \sqrt{x}]$ 段 | 与 $D_2$ 的 $D_{21}$ **关于 $y=x$ 对称** |

#### 5.2 累加

由对称性,**$D_{32}$ 上的积分 = 0**。

#### 5.3 判定 $D_{31}$ 上 $f$ 的符号

$D_{31}$ 上 $\sqrt{x} < y \leqslant 1$,即 $y > \sqrt{x} \geqslant x$(在 $[0,1]$ 内 $x \leqslant \sqrt{x}$)。所以 $y > x$ → $x - y < 0$ → $\sqrt[3]{x-y} < 0$。

由保号性:
$$J_3 = \iint_{D_{31}} \sqrt[3]{x-y}\,\mathrm{d}\sigma < 0$$

> 教材原话 (full.md:128-133):由保号性知,$J_3 = \iint_{D_{31}} \sqrt[3]{x-y}\,\mathrm{d}x\mathrm{d}y < 0$。

### 6. 结论

$$J_3 < 0 = J_1 < J_2$$

> **教材答案 (full.md:135)**:$J_3 < J_1 < J_2$,**应选 (B)**

### 7. ⚠️ 老王方法总结(必背)

| 子题 | 用什么 | 为什么 |
|------|--------|--------|
| $J_1$ | 普通对称性 + 偶倍奇零 | $D_1$ 本身关于 $y=x$ 对称,$f$ 满足奇函数条件 |
| $J_2, J_3$ | 普通对称性 + **辅助曲线** + 保号性 | $D_2, D_3$ 自身不对称 → 用 $y=x^2$ 或 $y=\sqrt{x}$ 创造对称性 |

> [!important] 老王原话 ⚠️
> "这种题**不要真去算积分**——它是性质问题,不是计算问题。四两拨千斤:画好辅助线,用好对称性 + 保号性,结果就出来了。"

## 三、轮换对称性

### 1. 引例 1(理解"积分值与用什么字母表示无关")

**问题**:以下两个二重积分是否相等?

$$\iint_{D_1:\, x^2/4 + y^2/3 \leqslant 1} (2x^2 + 3y^2)\,\mathrm{d}x\mathrm{d}y \stackrel{?}{=} \iint_{D_2:\, y^2/4 + x^2/3 \leqslant 1} (2y^2 + 3x^2)\,\mathrm{d}y\mathrm{d}x$$

#### 1.1 答案:相等 ✅ 教材核对 (full.md:161)

**老王原话**:"**积分值与用什么字母表示是无关的**——也就是把 $x$ 改成 $v$,把 $v$ 改成 $x$,你只是换了表示的字母,客观物体(那个大面包)**没动**,所以体积不变。"

### 2. 引例 2(理解"区域 $D$ 不变"这一关键条件)

**问题**:当区域**不变**(都是 $\frac{x^2}{4} + \frac{y^2}{4} \leqslant 1$)时,

$$\iint_{D:\, x^2/4 + y^2/4 \leqslant 1} (2x^2 + 3y^2)\,\mathrm{d}x\mathrm{d}y \stackrel{?}{=} \iint_{D:\, x^2/4 + y^2/4 \leqslant 1} (2y^2 + 3x^2)\,\mathrm{d}y\mathrm{d}x$$

**答案**:相等 ✅ 教材核对 (full.md:171)。理由同上。

#### 2.1 关键观察

把 $D$ 的边界 $x^2/4 + y^2/4 \leqslant 1$ 中的 $x,y$ **对调**,得到 $y^2/4 + x^2/4 \leqslant 1$,**完全相同**。所以 **$D$ 关于 $y=x$ 对称**。

**直观图示**:取单位圆 $x^2+y^2 \leqslant 1$ 为例,圆上两点 $(a,b)$ 与 $(b,a)$ 关于 $y=x$ 对称——**把 $D$ 上每个点的 $(x,y)$ 换成 $(y,x)$,新区域仍然是同一个 $D$**(圆不变)。

![Fig 4: 轮换对称性 -- 单位圆关于 y=x 对称](assets/fig4_rotational_symmetry.png)

### 3. 轮换对称性的严格定义

> [!tip] 轮换对称性使用条件 #重要 
> 在直角坐标系下,若把 $x$ 与 $y$ 对调后,**区域 $D$ 不变**(或区域 $D$ 关于 $y=x$ **对称**),则

$$\boxed{\iint_{D} f(x,y)\,\mathrm{d}\sigma = \iint_{D} f(y,x)\,\mathrm{d}\sigma}$$

其中 $\mathrm{d}\sigma = \mathrm{d}x\mathrm{d}y = \mathrm{d}y\mathrm{d}x$(乘法交换律)。

> [!warning] **必须是直角坐标系**
> 在其他坐标系(极坐标、椭圆坐标等)下没有"$\mathrm{d}\sigma = \mathrm{d}\sigma$"的交换律保证,所以**轮换对称性只在直角坐标系下成立**。

### 4. 抽象出来的核心恒等式 #重要 

由定义式立刻得到:

$$\iint_{D} f(x,y)\,\mathrm{d}\sigma = \frac{1}{2} \iint_{D} \bigl[f(x,y) + f(y,x)\bigr]\,\mathrm{d}\sigma$$

> [!tip] 老王原话 ⚠️
> "**单独一个** $f(x,y)$ 或者**单独一个** $f(y,x)$ 都不好算,但是**两者加起来** $f(x,y) + f(y,x)$ 经常变得简单。这才是轮换对称性真正的用途!"

区间再现公式：[定积分的计算](1-4.%20一元函数积分学/2.%20计算/定积分的计算.md#^rfcxg2)
### 5. 经典应用场景

> 教材原话 (full.md:201-205):在直角坐标系中,若 $f(x,y) + f(y,x) = a$,则

$$I = \frac{1}{2} \iint_{D} \bigl[f(x,y) + f(y,x)\bigr]\,\mathrm{d}x\mathrm{d}y = \frac{1}{2} \iint_{D} a\,\mathrm{d}x\mathrm{d}y = \frac{a}{2} S_D$$

#### 5.1 典型例子(教材 full.md:191 原例)

设 $D = \{(x,y) \mid x^2 + y^2 \leqslant 1,\ x>0,\ y>0\}$(第一象限单位圆),求

$$\iint_{D} \frac{a\sqrt{f(x)} + b\sqrt{f(y)}}{\sqrt{f(x)} + \sqrt{f(y)}}\,\mathrm{d}\sigma$$

**Step 1**:换字母。利用 $D$ 关于 $y=x$ 对称 + $\mathrm{d}x\mathrm{d}y = \mathrm{d}y\mathrm{d}x$,得:

$$\iint_{D} \frac{a\sqrt{f(x)} + b\sqrt{f(y)}}{\sqrt{f(x)} + \sqrt{f(y)}}\,\mathrm{d}\sigma = \iint_{D} \frac{a\sqrt{f(y)} + b\sqrt{f(x)}}{\sqrt{f(y)} + \sqrt{f(x)}}\,\mathrm{d}\sigma$$

**Step 2**:两个相加除以 2:

$$\iint_{D} \frac{a\sqrt{f(x)} + b\sqrt{f(y)}}{\sqrt{f(x)} + \sqrt{f(y)}}\,\mathrm{d}\sigma = \frac{1}{2}\iint_{D} \frac{(a+b)\bigl[\sqrt{f(x)} + \sqrt{f(y)}\bigr]}{\sqrt{f(x)} + \sqrt{f(y)}}\,\mathrm{d}\sigma = \frac{1}{2}(a+b) S_D$$

**Step 3**:$D$ 是**第一象限单位圆**,面积 $S_D = \dfrac{\pi \cdot 1^2}{4} = \dfrac{\pi}{4}$。

**最终**(教材 full.md:194 原文):
$$\iint_{D} \frac{a\sqrt{f(x)} + b\sqrt{f(y)}}{\sqrt{f(x)} + \sqrt{f(y)}}\,\mathrm{d}\sigma = \frac{a+b}{2} \cdot \frac{\pi}{4} = \frac{(a+b)\pi}{8}$$

> 教材原话 (full.md:191-195):这就是经典的轮换对称性化简。注意 $f$ 是**正值连续函数**(保证分母不为零)。

### 6. ⚠️ 普通对称性 ④ 与轮换对称性的对比(老王强调)

> 教材原话 (full.md:207):虽然它们都是 $D$ 关于 $y=x$ 对称,但...

| 维度 | 普通对称性 ④ | 轮换对称性 |
|------|---------------|------------|
| **前提** | $D$ 关于 $y=x$ 对称 | $D$ 关于 $y=x$ 对称 |
| **考查方式** | 看 $f(x,y)$ 与 $f(y,x)$ **相等还是相反** | 看 $f(x,y) + f(y,x)$ **是否简单** |
| **结论** | 相等 → 2 倍;相反 → 0 | **相加再除 2**,化简计算 |
| **用在哪** | $f$ 与 $f$ 对调后**关系清楚**(相等/相反) | $f$ 与 $f$ 对调后**关系不清楚**(既不相等也不相反) |
| **特例** | 当 $f(x,y) = -f(y,x)$ 时,两者**等价**(因为此时 $f(x,y) + f(y,x) = 0$,两边都是 0) | 当 $f(x,y) = -f(y,x)$ 时,两者**等价** |

> [!important] 选哪个?
> - **能判断 $f$ 与 $f(y,x)$ 关系**(相等/相反) → 用**普通对称性**(快)
> - **判断不了关系**(又相等又相反/没关系/超复杂) → 看 $f(x,y) + f(y,x)$ 是否简单 → 用**轮换对称性**

## 四、例 14.4:轮换对称性实战

> 教材原题 (full.md:209)

### 1. 题目 ✅ 教材核对 (full.md:209)

设
$$f(x) = \iint_{D(x)} \frac{v \ln\sqrt{u^2+v^2}}{u+v}\,\mathrm{d}u\mathrm{d}v$$

其中
$$D(x) = \left\{(u,v) \;\middle|\; \frac{1}{4} \leqslant u^2 + v^2 \leqslant x^2,\ u>0,\ v>0 \right\} \quad \left(x > \frac{1}{2}\right)$$

求 $f(x)$ 等于( )。

(A) $\dfrac{1}{4} \displaystyle\iint_{D(x)} \ln(u^2+v^2)\,\mathrm{d}u\mathrm{d}v$
(B) $\dfrac{1}{2} \displaystyle\iint_{D(x)} \ln(u^2+v^2)\,\mathrm{d}u\mathrm{d}v$
(C) $\displaystyle\iint_{D(x)} \ln(u^2+v^2)\,\mathrm{d}u\mathrm{d}v$
(D) $2 \displaystyle\iint_{D(x)} \ln(u^2+v^2)\,\mathrm{d}u\mathrm{d}v$

### 2. 分析(老王"程序"思路)

#### 2.1 第一步:看 $D$ 关于谁对称

$D(x) = \{(u,v) \mid \frac{1}{4} \leqslant u^2+v^2 \leqslant x^2,\ u>0,\ v>0\}$ 是**第一象限的圆环**:外圈半径 $x$,内圈半径 $1/2$。

- 把 $u, v$ 对调:$D(x)$ 的描述**不变**(因为 $u^2+v^2 = v^2+u^2$,$u>0, v>0$ 也不变)
- 所以 $D(x)$ **关于 $y=x$ 对称** ✅ 教材核对 (full.md:211)

#### 2.2 第二步:看被积函数的关系

设 $g(u,v) = \dfrac{v \ln\sqrt{u^2+v^2}}{u+v}$。

- 代入对称点 $(v,u)$:
$$g(v,u) = \frac{u \ln\sqrt{v^2+u^2}}{v+u} = \frac{u \ln\sqrt{u^2+v^2}}{u+v}$$

- 关系: $g(u,v) \neq g(v,u)$(分子一个是 $v$ 一个是 $u$),$g(u,v) \neq -g(v,u)$(都是正的)
- 既**不相等也不相反** → **普通对称性用不上**

#### 2.3 第三步:考虑轮换对称性(看 $g(u,v) + g(v,u)$)

$$g(u,v) + g(v,u) = \frac{(u+v)\ln\sqrt{u^2+v^2}}{u+v} = \ln\sqrt{u^2+v^2}$$

> [!tip] 化简成功!
> 分子分母中的 $u+v$ **约掉了**!分母只剩一个 $\ln\sqrt{u^2+v^2}$ —— 这就是为什么"轮换对称性是换字母之后**对交换后的 $F$ 进行运算**"。

### 3. 求解

$$f(x) = \iint_{D(x)} g(u,v)\,\mathrm{d}u\mathrm{d}v = \iint_{D(x)} g(v,u)\,\mathrm{d}u\mathrm{d}v \quad \text{(轮换对称性)}$$

$$2f(x) = \iint_{D(x)} \bigl[g(u,v) + g(v,u)\bigr]\,\mathrm{d}u\mathrm{d}v = \iint_{D(x)} \ln\sqrt{u^2+v^2}\,\mathrm{d}u\mathrm{d}v$$

$$f(x) = \frac{1}{2} \iint_{D(x)} \ln\sqrt{u^2+v^2}\,\mathrm{d}u\mathrm{d}v = \frac{1}{2} \iint_{D(x)} \frac{1}{2}\ln(u^2+v^2)\,\mathrm{d}u\mathrm{d}v = \frac{1}{4} \iint_{D(x)} \ln(u^2+v^2)\,\mathrm{d}u\mathrm{d}v$$

> 教材答案 (full.md:213):**应选 (A)**

### 4. ⚠️ 易错点提醒

> [!warning] 易错点 ⚠️
> 1. **别忘了 $\ln\sqrt{\cdot} = \frac{1}{2}\ln(\cdot)$**:这一半的系数直接决定了 (A) 还是 (B) 还是 (C)。老王这里转换了**两次**:$\ln\sqrt{u^2+v^2} = \dfrac{1}{2}\ln(u^2+v^2)$,所以 $\frac{1}{2} \times \frac{1}{2} = \frac{1}{4}$,正好对应选项 (A)。
> 2. **积分区域 $D(x)$ 是动态的**:外圈半径 $x$ 是变量,内圈半径 $1/2$ 是常数。这种"圆环"的轮换对称性跟"圆"一样,因为**换字母后 $D(x)$ 的表达式不变**。
> 3. **第一象限条件** $u>0, v>0$:保证 $u+v > 0$,分母不为零。

### 5. 方法总结(老王总结)

> 教材原话 (full.md:221):"当 $D$ 关于 $y=x$ 对称,$f(x,y) = -f(y,x)$ 或 $f(x,y) = f(y,x)$ 时,考虑普通对称性;若不满足,可看 $f(x,y) + f(y,x)$ 是否简单,考虑轮换对称性。"

**轮换对称性的标准流程**:

| Step | 动作 |
|------|------|
| **Step 1** | 判断 $D$ 是否关于 $y=x$ 对称(把 $D$ 的描述 $x, y$ 互换后**表达式不变**) |
| **Step 2** | 判断 $f(x,y)$ 与 $f(y,x)$ 是否**相等/相反** |
| **Step 2a** | 若能判断 → 用**普通对称性**(快) |
| **Step 2b** | 若不能判断 → 看 $f(x,y) + f(y,x)$ 是否**简单** |
| **Step 3** | 若简单 → 用**轮换对称性**:$I = \frac{1}{2}\iint_D [f(x,y)+f(y,x)]\,\mathrm{d}\sigma$ |

## 六、本节易错点速查

| 编号 | 易错点 | 正确做法 |
|------|--------|----------|
| 1 | 把多元函数的"奇/偶"当作 1 元的定义 | 多元**没有**奇偶函数的定义,只是借用口诀,**实质是代入对称点判关系** |
| 2 | 对称轴是 $x=a$ 时,对称点忘了加 $a$ | 关于 $x=a$ 的对称点是 $(2a-x, y)$,**不是** $(-x, y)$ |
| 3 | 轮换对称性在极坐标下用 | 轮换对称性**只适用于直角坐标系** |
| 4 | 例 14.3 真去算积分 | 比大小**不需要**真算,用对称性 + 保号性即可 |
| 5 | 例 14.4 漏掉 $\ln\sqrt{\cdot} = \frac{1}{2}\ln(\cdot)$ 的系数 | 这一半系数直接决定选项 |
| 6 | 例 14.4 用普通对称性 ④ 而忽略 $f$ 跟 $f(y,x)$ 的关系 | $g(u,v)$ 跟 $g(v,u)$ **既不相等也不相反**,必须用轮换对称性 |
| 7 | 写 $D$ 关于 $y=u$ 对称 | 应该是 $D$ 关于 $y=x$ 对称(讲课稿口语"YU=V"是 "$u=v$" 即 $y=x$ 的 ASR 转写) |
