## 二、直角坐标-直线边界(教材 D1-D17,full.md:9-77)

### 1. 入门:最朴素的几块

**例 (1)**:$D = \{(x,y) \mid 0 \le x \le 1,\ 0 \le y \le 1\}$
→ 单位正方形 ✅
![[二重积分区域D的画图(三大体系)-1783742462758.webp|291]]
**例 (2)**:$D = \{(x,y) \mid x+y \le 1,\ x \ge 0,\ y \ge 0\}$
→ 第一象限的直角三角形
![[二重积分区域D的画图(三大体系)-1783742481900.webp|297]]
### 2. 斜线情形 ⚠️ 易错

**例 (4)**:$D = \{(x,y) \mid 0 \le x \le 2,\ x \le y \le \sqrt{3}\,x\}$
![[二重积分区域D的画图(三大体系)-1783742509890.webp|275]]

| 边 | 含义 |
|----|------|
| $y=x$ | 角度 $\pi/4$ 的射线 |
| $y=\sqrt{3}\,x$ | 角度 $\pi/3$ 的射线 |
| $0\le x\le 2$ | 极径 0 到 2 之间一块 |

→ **第一象限里以原点为顶点、张角 $\pi/12$ 的扇形再截掉 $x\le 2$ 的部分**

**例 (7)**:$D = \{(x,y) \mid 0 \le x \le 3,\ 0 \le y \le 3,\ \tfrac{\sqrt{3}}{3}x \le y \le \sqrt{3}x\}$

$\tfrac{\sqrt{3}}{3} = \tan(\pi/6)$,$\sqrt{3} = \tan(\pi/3)$ —— 角度 $\pi/6$ 到 $\pi/3$ 的斜条带。

### 3. 复合不等式(线之间夹一块)

**例 (6)**:$D = \{(x,y) \mid 1 \le x+y \le 2,\ x \ge 0,\ y \ge 0\}$

两条平行线 $x+y=1$ 与 $x+y=2$ 在第一象限截出来的**梯形/平行四边形**。
![[二重积分区域D的画图(三大体系)-1783742531250.webp|287]]
**例 (9)**:$D = \{(x,y) \mid 0 \le x+y \le 1,\ 0 \le y \le 1\}$

横线 $y=0,1$ 与斜线 $x+y=0,1$ 围出来的**四角形**(顶点在原点)。
![[二重积分区域D的画图(三大体系)-1783742554493.webp|333]]
### 4. 绝对值专题 ⚠️⭐ 老王重点 #重要 

> 老王原话:**"在直角系下的直线这个直线边界里头,我特别强调一下,就是关于这个什么绝对值的"**

**例 (15)**:$D = \{(x,y) \mid |x| + |y| \le 1\}$
![[二重积分区域D的画图(三大体系)-1783742577065.webp|251]]
分四个象限去绝对值:

| 象限 | 去绝对值后 | 边界线 |
|------|----------|--------|
| 第一($x,y \ge 0$) | $x+y \le 1$ | 直线 $x+y=1$ |
| 第四($x \ge 0,y \le 0$) | $x-y \le 1$ | 直线 $x-y=1$ |
| 第二($x \le 0,y \ge 0$) | $-x+y \le 1$ | 直线 $-x+y=1$ |
| 第三($x,y \le 0$) | $-x-y \le 1$ | 直线 $-x-y=1$ |

→ 围出**斜 45° 放置的正方形**。

> [!warning] 易错点
> 绝对值的画图需要**先分象限 / 分区间去绝对值**,再画线,不要凭想象画。

**例 (16)**:$D = \{(x,y) \mid 0 \le x \le 1,\ 0 \le y \le 1,\ |x-y| \le \tfrac{1}{2}\}$
![[二重积分区域D的画图(三大体系)-1783742598614.webp|229]]
去绝对值后是 $-\tfrac{1}{2} \le x-y \le \tfrac{1}{2}$,在单位正方形里两条平行线 $y = x \pm \tfrac{1}{2}$ 之间夹出来的一条**斜带**。

**例 (17)**:$D = \{(x,y) \mid |x| \le 1,\ |y| \le 1\}$
→ 中心在原点、边长 2 的正方形。
![[二重积分区域D的画图(三大体系)-1783742619097.webp|259]]
### 5. 描点示意 vs 直接画图 ⚠️ 秒杀技巧

> 老王原话:**"看着这个题这个表达式,你就把这个图像给我画出来"**

凡是遇到这种训练,**画法永远只有一个顺序**:

1. **去绝对值** / 拆复合不等号
2. **把每条边界画出来**
3. **看方向**($\le$ 取这一侧,$\ge$ 取那一侧)
4. **连起来**

## 三、直角坐标-曲线边界(教材 D1-D35,full.md:81-269)

### 1. 单曲线 + 直角坐标框

**例 (1)**:$D = \{(x,y) \mid 0 \le x \le 1,\ \sqrt[3]{x} \le y \le 1\}$
![[二重积分区域D的画图(三大体系)-1783742638813.webp|240]]
- $x=0$ 左边界,$x=1$ 右边界
- $y=1$ 上边界,$y=\sqrt[3]{x}$ 下边界

→ 第一象限内,$x \in [0,1]$ 时,$y$ 从 $\sqrt[3]{x}$ (经过 (0,0)) 到 1。

**例 (2)**:$D = \{(x,y) \mid 0 \le x \le 1,\ \arctan x \le y \le \pi/4\}$
![[二重积分区域D的画图(三大体系)-1783742666047.webp|254]]
- 下界 $y = \arctan x$ (从 (0,0) 到 (1, π/4))
- 上界 $y = \pi/4$ 水平线

→ 第一象限一个弓形(下半由 arctan 包,上半由水平线封顶)。

### 2. 圆 + 直线组合 ⚠️⭐ 真题高频

**例 (7)**:$D = \{(x,y) \mid (x-1)^2 + (y-1)^2 \le 2,\ 0 \le x+y \le 4\}$
![[二重积分区域D的画图(三大体系)-1783742687130.webp|250]]
- 圆心 (1,1)、半径 $\sqrt{2}$ 的圆
- 加 $0 \le x+y \le 4$ 两条平行线

老王原话讲的关键:**"$x+y=0$ 过原点的就是这条线的切线"** —— 通过几何验证:$(0,0)$ 到圆心 $(1,1)$ 距离 $\sqrt{2}$,而半径也是 $\sqrt{2}$,所以 $(0,0)$ 恰在圆上 → $x+y=0$ 是**过原点的切线**(切点就是 (0,0))。

> [!warning] 老王的几何判断法
> **识别切线**:已知"圆心到直线的距离 = 半径"就是切线。本例:$d(O, x+y=0) = \sqrt{2}$ = 半径,所以是切线。

### 3. 反例/分情况题 ⚠️ 难度较高 #困难 

**例 (8)**:$D = \{(x,y) \mid \tfrac{1}{4} \le x^2+y^2,\ x^2+y^2 \ge x^4+y^4\}$
![[二重积分区域D的画图(三大体系)-1783742709948.webp|284]]
老王的两种看法:

**看法 A (代数化简)** 

$$x^4+y^4 \le x^2+y^2$$

等价变形 —— 老王把 $(x^4+y^4)$ 改写成 $(x^2+y^2)^2 - 2x^2y^2$,所以不等式变成 $(x^2+y^2)^2 - 2x^2y^2 \le x^2+y^2$ → 提取公因子和配方:

$$(x^2+y^2)^2 \le (x^2+y^2) + 2x^2y^2 = (x^2+y^2)(1 + \tfrac{2x^2y^2}{x^2+y^2})$$

更锋利的看法:两边乘 $(x^2+y^2)$ 取**开方**(老王说"$x^2+y^2$ 是小榆等号小于等于某个量"):

$$\sqrt{(x^2+y^2)^2} \le \sqrt{(x^2+y^2)^2 + 2x^2y^2} \Rightarrow x^2+y^2 \le \sqrt{(x^2+y^2)^2 + 2x^2y^2}$$

这告诉我们 **$x^2+y^2$ 在那条曲线** 上是有上界的。但**这条曲线的表达式并不简单**。老王承认"我不写精化推导,只要拿到几个**关键点**就够了"。

**看法 B (描点法)**:

- 取 $y = x$ 上的点 → $2x^4 = 2x^2$ → $x^2=1$ → $x = \pm 1$
- 取 $x = 0$ 的点 → $y^4 = y^2$ → $y = \pm 1$
- 取 $y = 0$ 的点 → $x^4 = x^2$ → $x = \pm 1$

→ 4 个交点 (1,1),(1,-1),(-1,-1),(-1,1) 都满足,加上内点约束。

**看法 C (转极坐标)**:教材讲法

令 $x^2+y^2 = r^2$,$x^2 y^2 = r^4 \cos^2\theta \sin^2\theta$,代入 $x^4+y^4$:

$$\begin{aligned}
(x^2+y^2)^2 - 2x^2y^2 &\le x^2+y^2 \\
(x^2+y^2)(x^2+y^2 - 1) &\le 2x^2y^2
\end{aligned}$$

令 $x^2+y^2 = r^2$,$2x^2y^2 = 2r^4\cos^2\theta\sin^2\theta = \tfrac{1}{2}r^4\sin^2 2\theta$,得:

$$\boxed{r^2\big(r^2 - 1\big) \le \tfrac{1}{2}\, r^4 \sin^2 2\theta}$$

 老王承认这部分"我写不出来精化,只取几个特殊的 $\theta$ 描几个点就够了"——所以走**看法 B 描点**更快。

> ✅ 例 (8) 表达式已与教材 (full.md:117) 核对。

### 4. 双纽线 ⚠️⭐ 老王重点讲

**例 (12)**: $D = \{(x,y) \mid (x^2+y^2)^2 \le 2xy,\ x \ge 0,\ y \ge 0\}$

![[二重积分区域D的画图(三大体系)-1783742759650.webp|221]]

化到极坐标:$(r^2)^2 \le 2r^2\cos\theta\sin\theta$,即 $r^2 \le \sin 2\theta$,即 $\boxed{r \le \sqrt{\sin 2\theta}},\ \theta \in [0,\pi/2]$。

**画图四步** (老王口述):

1. **画 $\sin 2\theta$ 在 $\theta \in [0,\pi/2]$ 的形状**:2 倍角意味着 $\theta$ 横坐标"压缩一半" → 实际覆盖 $[0,\pi]$(对应 $2\theta$ 的一个完整正弦瓣)。极值在 $\theta = \pi/4$ 处 $r = 1$。两端 $\theta=0$ 和 $\theta=\pi/2$ 处 $r=0$。
2. **$\sqrt{\cdot}$ 单调递增** → 不改变零点/最大值位置
3. **围出来的区域是**:极点在 (0,0),在 $\theta=\pi/4$ 方向"鼓出最远",半径为 1,左右两瓣夹在第一象限。
4. → **只在第一象限的双纽线瓣** (类似四叶玫瑰的一瓣)

> [!tip] 秒杀技巧
> 凡是看到 $(x^2+y^2)^2 \le kxy$ 这种**四次齐次式**,**直接转极坐标** $r = \sqrt{\sin 2\theta}$ 之类,不用在直角坐标死磕。

**例 (19)**:$D = \{(x,y) \mid (x^2+y^2)^2 \le 2xy\}$ —— 全平面版的(无 $x,y\ge0$ 限制),**双纽线**(类似 $\infty$ 形,横躺/竖躺要看 $xy$ 还是 $x^2-y^2$)。
![[二重积分区域D的画图(三大体系)-1783742949854.webp|273]]
### 5. 凹凸性判断 ⚠️ 老王密传

**例 (16)**: $D = \{(x,y) \mid x+y+xy \ge 1,\ x^2+y^2 \le 1,\ x \ge 0,\ y \ge 0\}$

![[二重积分区域D的画图(三大体系)-1783742982272.webp|304]]

老王方法:**隐函数求导判凹凸**(不要怕):

$$\text{对 }x+y+xy=1\text{ 求导} \Rightarrow 1 + y' + y + xy' = 0$$

$$\Rightarrow y'(1+x) = -(1+y) \Rightarrow y' = -\frac{1+y}{1+x}$$

**二阶导**:

$$y'' = -\frac{y'(1+x) - (1+y)}{(1+x)^2} = \frac{2(1+y)}{(1+x)^2 \cdot \text{(某一系数)}}$$

老王判断:在 $x \in [0,1]$ 时,$y' < 0$ (单调减),$y'' > 0$ (凹曲线)。

> [!tip] 老王口诀
> "**见到复杂曲线不要怕,描点 + 对称性 + 连续曲线顺势连,就能搞定**"。

## 四、极坐标(教材 D1-D15 + 注的真题,full.md:275-353)

### 1. 简单极坐标 (教材对照表)

| 教材编号 | 极坐标区域 | 是什么 |
|--------|-----------|--------|
| (1) | $0 \le r \le \csc\theta,\ \pi/4 \le \theta \le \pi/2$ | 第一象限里,$r = \csc\theta$ 即 $y = 1$ 的水平线下方到原点 |
| (2) | $0 \le r \le \sec\theta,\ 0 \le \theta \le \pi/4$ | $r = \sec\theta$ 即 $x = 1$ 的垂线右侧,$\theta \in [0,\pi/4]$ |
| (3) | $0 \le \theta \le 2\pi,\ \theta/2 \le r \le \pi$ | 阿基米德螺线一块 |
| (4) | $0 \le r \le 1,\ 0 \le \theta \le \pi/4$ | 半径 1 的扇形 |
| (7) | $0 \le r \le \sqrt{\sin 2\theta},\ 0 \le \theta \le \pi/2$ | **双纽线一瓣**(同 §三例 12)|
| (11) | $0 \le r \le 2\sin\theta,\ 0 \le \theta \le \pi/4$ | 在圆 $r=2\sin\theta$ 内部,角度切片 |

> [!warning] 极坐标转换桥梁(必背)
> $$\boxed{\begin{cases} x = r\cos\theta \\ y = r\sin\theta \end{cases}} \Rightarrow \boxed{x^2+y^2 = r^2,\quad \frac{y}{x} = \tan\theta}$$
>
> 反向:**看到 $R=\sin\theta$ → $y = r\sin\theta \Rightarrow y = \dfrac{R\sin\theta}{1} \Rightarrow y = 1$** 类似手法。

### 2. 真题(教材 2019 数学二 18 题) ⚠️⭐⭐⭐ 高频

教材注 (full.md: 347-353):

![[二重积分区域D的画图(三大体系)-1783754510829.webp|305]]

> 有些 $D$ 是用直角坐标系下的表达式描述的,但实际上用极坐标方程更方便,要归到这里来。如 **2019 年考研真题数学二(18)** 的平面区域为
> $$D = \{(x,y) \mid |x| \le y,\ (x^2+y^2)^3 \le y^4\}$$

**老王解法三步**:

**Step 1**:转极坐标。

- $|x| \le y \Rightarrow r|\cos\theta| \le r\sin\theta \Rightarrow |\cos\theta| \le \sin\theta$
- $(x^2+y^2)^3 \le y^4 \Rightarrow r^6 \le r^4 \sin^4\theta \Rightarrow r^2 \le \sin^4\theta$

**Step 2**:分析 $\theta$ 范围。
$|\cos\theta| \le \sin\theta$,因为 $\sin\theta > 0$ 才能让 $\sin\theta$ 大过 $|\cos\theta|$,所以 $\theta \in [0, \pi]$。再分两部分:

- $0 \le \theta \le \pi/2$ 时,$\cos\theta \le \sin\theta \Rightarrow \tan\theta \ge 1 \Rightarrow \theta \ge \pi/4$
- $\pi/2 \le \theta \le \pi$ 时,$-\cos\theta \le \sin\theta$ 自动成立

→ 所以 $\theta \in [\pi/4, \pi]$。

**Step 3**:化 $r^2 = \sin^4\theta$ → $r = \sin^2\theta = \left(\tfrac{1-\cos 2\theta}{2}\right)^2$,画图。

> ✅ 2019 真题表达式已与教材 (full.md:347-353) 核对。
> 老王强调:**"很多同学这图画不出来,这个图当年考的时候很多人都不会,你会上转化,从这一步到这一步就很简单了"**。

## 五、参数方程(教材 D1-D9,full.md:359-398)

### 1. 动边界型 ⚠️⭐ 老王重点

**例 (1)**: $D = \{(x,y) \mid 0 \le y \le t - x,\ 0 \le x \le t,\ 0 < t \le 1\}$

![[二重积分区域D的画图(三大体系)-1783743126120.webp|301]]

- $x = 0$ 到 $x = t$
- $y = 0$ 到 $y = t - x$(随 $t$ 变化的斜线下边界)
- $t \in (0, 1]$ 时,$D$ 是个**变面积直角三角形**(斜边在动)

**例 (2)**: $D = \{(x,y) \mid 0 \le x \le t,\ x \le y \le t\}$

![[二重积分区域D的画图(三大体系)-1783743149552.webp|281]]

- $x = 0$ 到 $x = t$
- $y$ 从 $y = x$ 到 $y = t$,上边界水平,下边界斜
- → 变梯形/三角形

### 2. 圆 + 椭圆临界情形 ⚠️⭐⭐ 真题级难度

**例 (5)**: $D = \{(x,y) \mid x^2 + (y-1)^2 \le 1,\ \tfrac{2}{3}x^2 + \tfrac{2}{9}y^2 \le a^2,\ a > 0\}$

![[二重积分区域D的画图(三大体系)-1783743178025.webp|597]]

- 圆 $x^2 + (y-1)^2 \le 1$ — **固定不动**(圆心 (0,1)、半径 1)
- 椭圆 $\tfrac{2}{3}x^2 + \tfrac{2}{9}y^2 \le a^2$ — **大小随 $a$ 变化**
- 椭圆公式： $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$

**求临界 $a$**(老王讲法):

> 老王原话:**"找临界位置,临界位置的话,就是切在里面,就在这个位置相切"**

**相切条件**:在 $x = 0$ 时,圆上的 $y$ 值和椭圆上的 $y$ 值要相等。

- 圆 $x=0$:$(y-1)^2 = 1 \Rightarrow y = 0$ 或 $y = 2$。取 $y = 2$(上半圆)。
- 椭圆 $x=0$:$\tfrac{2}{9}y^2 = a^2 \Rightarrow y = \tfrac{3a}{\sqrt{2}}$(取正)。

令 $\tfrac{3a}{\sqrt{2}} = 2 \Rightarrow a^2 = \tfrac{8}{9} \Rightarrow \boxed{a = \tfrac{2\sqrt{2}}{3}}$。

**老王判断**:

- $a > \tfrac{2\sqrt{2}}{3}$:椭圆把圆**包起来**
- $a < \tfrac{2\sqrt{2}}{3}$:圆和椭圆**相交**(圆在椭圆外伸出一部分)
- $a = \tfrac{2\sqrt{2}}{3}$:**内切**

> [!tip] 老王原话总结
> **"大于临界 → 把包起来;小于临界 → 露出来;等于临界 → 切着"**

## 六、老王方法论:"做题 = 画图"

### 1. 三步画图法

老王总结:拿到一个表达式,画图的标准动作:

```
1. 拆界限(去绝对值 / 分象限 / 分区间)
2. 描点(取几对特殊 (x,y) 验一下)
3. 连线(连续曲线顺势连 — 不要死磕精确位置)
```

### 2. 描点法万能

> 老王原话:**"见到这种曲线,你一定不要怕,描点法,你连续曲线那肯定连曲线了"**

例:画 $x+y+xy = 1$,随便取几个点:

- $x=0 \Rightarrow y = 1$
- $y=0 \Rightarrow x = 1$
- $x = y \Rightarrow 2x+x^2 = 1 \Rightarrow x = \sqrt{2} - 1$ ... 这种点不太好算,但前两个够画了。

### 3. 对称性

- **关于 $y = x$ 轮换对称**:表达式里 $x,y$ 互换不变。
- **关于原点对称**:$(x,y) \to (-x,-y)$ 不变。
- **关于 $x$ / $y$ 轴对称**:把另一个变量取负不变。

### 4. 老王反复强调的态度 ⚠️

> 老王原话:**"这个画法是大概齐,只是个大致的情况,你大致就行了,是不是?大概是什么样的图,你描点法就可以"**

**考场上不要追求精确画图** —— 画对大致形状 + 抓住关键交点 + 知道"哪一侧是 D",够用。

## 七、基础习题(教材 full.md:404-413)

### 习题 14.1(累次积分化二重积分的逆运算) ⚠️ 教材核对 (full.md:404)

$$\lim_{n \to \infty} \sum_{i=1}^n \sum_{j=1}^n \frac{n}{(n+i)(n^2+j^2)} = (?)$$

| 选项 | 表达式 |
|------|--------|
| (A) | $\int_0^1 dx \int_0^x \dfrac{1}{(1+x)(1+y^2)} dy$ |
| (B) | $\int_0^1 dx \int_0^x \dfrac{1}{(1+x)(1+y)} dy$ |
| (C) | $\int_0^1 dx \int_0^1 \dfrac{1}{(1+x)(1+y)} dy$ |
| (D) | $\int_0^1 dx \int_0^1 \dfrac{1}{(1+x)(1+y^2)} dy$ |

> ✅ 表达式已与教材 (full.md:404-408) 核对。
>
> **解题思路**(双元 Riemann 和 → 二重积分):
>
> 1. 分子分母同除 $n^3$,拆 $\dfrac{1}{n+i} = \dfrac{1/n}{1+i/n}$,$\dfrac{1}{n^2+j^2} = \dfrac{1/n^2}{1+(j/n)^2}$
>
>    $$\sum\sum \frac{n}{(n+i)(n^2+j^2)} = \sum\sum \frac{1/n^2}{(1+i/n)\big(1+(j/n)^2\big)}$$
>
> 2. 令 $x = i/n,\ y = j/n$:这是 $x\in[0,1], y\in[0,1]$ 的二重 Riemann 和,但 **$y$ 的分母是 $1+y^2$ 不是 $1+y$** → 排除 (C)(B)
> 3. 内外限:**$i$ 不依赖 $j$**,但 $j/n$ 也不依赖 $i/n$ → 这是**长方形区域 $[0,1]\times[0,1]$** 的二重积分,**不是**先 $y$ 从 $0$ 到 $x$ 的三角形积分 → 排除 (A)
>
> → **答案 = (D)** ⚠️ 老王没讲,纯推断

### 习题 14.2(对称性 + 极坐标判选项) ✅ 教材核对 (full.md:410)

设 $f(t)$ 连续,$D = \{(x,y) \mid x^2+y^2 \le 2y\}$,则

$$\iint_D f(xy)\, dx\,dy = (?)$$

| 选项 | 表达式 |
|------|--------|
| (A) | $\int_{-1}^1 dx \int_{-\sqrt{1-x^2}}^{\sqrt{1-x^2}} f(xy)\, dy$ |
| (B) | $2\int_0^2 dy \int_0^{\sqrt{2y-y^2}} f(xy)\, dx$ |
| (C) | $\int_0^\pi d\theta \int_0^{2\sin\theta} f(r^2\sin\theta\cos\theta)\, dr$ |
| (D) | $\int_0^\pi d\theta \int_0^{2\sin\theta} f(r^2\sin\theta\cos\theta)\, \color{red}{r}\, dr$ |

> ✅ (C)(D) 选项已与教材 (full.md:414) 核对。
>
> **判选项老三步**:
>
> 1. **画区域**:$D = \{(x,y) \mid x^2+y^2 \le 2y\} = \{(x,y) \mid x^2+(y-1)^2 \le 1\}$ → 圆心 (0,1)、半径 1 的圆,**不是**单位圆。直接排 **(A)**(A 默认上半单位圆 $|x|\le\sqrt{1-x^2}$)。
> 2. **轮换 / 普通对称性**:$f(xy)$ 关于**原点对称**($xy \to (-x)(-y) = xy$ 不变) + $D$ 关于 $y$ 轴对称 → 上下对称、左右对称,(B) 写的是 $2\int_0^2 dy \int_0^{\cdots} dx$ 这形式**对**但只对了**右半**。**关于 $y$ 对称意味着可以 $2\cdot$ 右半,但本积分方向是 $dy dx$,先 $x$ 后 $y$ 时关于 $y$ 对称没用**(此时先积的是 $x$,$x$ 关于 $y$ 轴不对称的性质不起作用,得手动写 $2\int$),(B) 写法勉强正确——但极坐标更清爽。
> 3. **极坐标必乘 $r$**:(C) 漏乘 $r$,(D) 乘了 → **答案 = (D)** ✅
>
> 老王这类题秒杀思路:**一画图,二对称性,三必乘 $r$ —— 必出答案**。
