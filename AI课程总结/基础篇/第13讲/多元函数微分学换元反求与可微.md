## 一、变限积分求偏导 · 换元型（例 13.5）

### 1. 题型识别：可拆分型 vs 换元型

上讲的例13.4讲过一种——被积函数能**把求导变量 $x$ 提出来**，剩下只有积分变量 $t$，这叫"**可拆分型**"。

本讲例13.5是另一种——被积函数里 $x$ 和 $t$ **死死粘在一起**（比如 $-xt^2$），提不出来，必须**换元**才能把 $x$ 弄到积分号外面去。

| 题型 | 识别标志 | 处理方法 |
|------|----------|----------|
| 可拆分型 | 求导变量 $x$ 能提到积分号外 | 直接套变限积分求导公式 |
| **换元型** | 求导变量 $x$ 和积分变量 $t$ 缠在一起 | **换元法**：$u = \sqrt{x}\cdot t$（或类似），把 $x$ 分离出去 |

### 2. 例 13.5 精讲

设 
$$f(x,y) = \int_0^{xy} e^{-xt^2}\,dt, \quad (x,y)\in(0,+\infty)\times(0,+\infty)$$
求 $f_x(1,+\infty)$。

> **易错提醒**：题目里的"+∞"是**广义点**——把它当作"很远处的 $y_0$"处理。由于 $x>0$（$x=1$ 附近走增量还是正），$y\to+\infty$ 也保持正，所以**待会儿的换元 $\sqrt{x}$ 是有意义的**。

#### 方法一：先代值再求导 ✅（秒杀推荐）

> [!tip] 代值带的是什么？
> 带入的应该是 y 的值，因为是对 x 求偏导，y 被看作常数，可以直接代值，**不是说将 x 的值带入**

**第一步**：把 $y=+\infty$（即 $y_0$）先代进去。

$$f(x, +\infty) = \int_0^{+\infty} e^{-xt^2}\,dt$$

注意 $x>0$，被积函数里 $x$ 还在，但**和 $t$ 绑在一起**，得换元。

**第二步**：凑高斯积分 $\displaystyle\int_0^{+\infty} e^{-u^2}\,du = \frac{\sqrt{\pi}}{2}$。[Gamma函数](../../../1-4.%20一元函数积分学/2.%20计算/Gamma函数.md)

原式有 $-xt^2 = -(\sqrt{x}\,t)^2$，令 $u = \sqrt{x}\,t$（$x$ 当常数看）：

| 换元步骤 | 操作 |
|----------|------|
| **被积函数换** | $e^{-xt^2}\,dt \to e^{-u^2}\,du$ |
| **积分元素换** | $dt = \dfrac{du}{\sqrt{x}}$（即 $\sqrt{\dfrac{1}{x}}\,du$）|
| **上下限换** | $t=0 \to u=0$；$t\to+\infty \to u\to+\infty$ |

> **避坑**：换元法"**三换**"——被积函数换、积分元素 $dt$ 换、上下限换，**一个不能少**。

代入：
$$f(x,+\infty) = \int_0^{+\infty} e^{-u^2}\cdot \sqrt{\frac{1}{x}}\,du = \sqrt{\frac{1}{x}}\cdot \frac{\sqrt{\pi}}{2}$$

**第三步**：对 $x$ 求导，再代 $x=1$。

$$f_x(x,+\infty) = \frac{\sqrt{\pi}}{2}\cdot \left(-\frac{1}{2}\right)x^{-3/2}$$

把 $x=1$ 代入：$f_x(1,+\infty) = -\dfrac{\sqrt{\pi}}{4}$。

#### 方法二：先求导后代值 ❌（复杂，不推荐）

> **避坑**：可以用，但**过程繁琐**——对 $\displaystyle\int_0^{xy} e^{-xt^2}\,dt$ 直接对 $x$ 求偏导，要把 $x$ 既当**积分上限里**的因子、又当**被积函数里**的因子分别处理。算到最后代入 $(1,+\infty)$ 也能得到 $-\dfrac{\sqrt{\pi}}{4}$（和方法一结果一样），但**计算量翻倍**。

具体地（了解即可）：
$$f_x = \int_0^{xy} \frac{\partial}{\partial x}\left(e^{-xt^2}\right)dt + e^{-x(xy)^2}\cdot(y) = -\int_0^{xy}t^2 e^{-xt^2}\,dt + y\,e^{-x^3y^2}$$

> **秒杀技巧**：
> - 见到 $f_x(x_0, y_0)$（**特定点**的偏导），**永远先想**："能不能先把另一个变量 $y=y_0$ 代进去？"
> - 代入后表达式变简单（一元函数），再对 $x$ 求导，**最后代 $x=x_0$**。
> - 这一招叫"**先代值再求导**"，是本讲的**最核心秒杀**。

### 3. 一句话总结本节

> 换元型变限积分求偏导，**流程是**：换元（凑已知积分）→ 化为一元函数 → 求导 → 代值。**先代值再求导**，速度翻倍。

## 二、反问题：已知偏导反求原函数（例 13.6）

### 1. 题型识别

题目给你 $\dfrac{\partial f}{\partial x}$、$\dfrac{\partial f}{\partial y}$，加上初始条件 $f(x_0,y_0)=C$，让你**反求** $f(x,y)$ 的表达式。

**本质**：多元积分的"反问题"，对应一元里的"已知 $F'(x)=f(x)$，求 $F(x)$"。

### 2. 一元 → 多元的"一脉相承 + 创新"

**一元情况**（复习）：
$$F'(x) = \cos x \implies F(x) = \int \cos x\,dx = \sin x + C$$

**二元情况**：
$$\frac{\partial f}{\partial y} = e^{-x}\cos y \implies f(x,y) = \int e^{-x}\cos y\,dy = e^{-x}\sin y + \varphi(x)$$

> **避坑（核心考点）**：
> - 一元加的是**任意常数 $C$**（一个数）。
> - 多元对 $y$ 积分时，$x$ 被**当成常数**，所以积分"常数"也得是 $x$ 的函数，记作 $\varphi(x)$（**关于 $x$ 的任意函数**）。
> - 反过来：对 $x$ 积分时，要加 $\psi(y)$（**关于 $y$ 的任意函数**）。

> **口诀**：对 $y$ 积分加 $\varphi(x)$，对 $x$ 积分加 $\psi(y)$。**加谁的反面**。

### 3. 例 13.6 精讲

设 $f(x,y)$ 可微，$f(0,0)=0$，且
$$\frac{\partial f}{\partial x} = -f, \qquad \frac{\partial f}{\partial y} = e^{-x}\cos y$$
求 $f(x,y)$。

**第一步**：从 $\dfrac{\partial f}{\partial y} = e^{-x}\cos y$ 入手（这条更直接，谁求导得 $\cos y$？答案是 $\sin y$）。

对 $y$ 积分（$x$ 当常数）：
$$f(x,y) = e^{-x}\sin y + \varphi(x)$$

**第二步**：利用 $\dfrac{\partial f}{\partial x} = -f$，对 $x$ 求偏导。

$$\frac{\partial f}{\partial x} = -e^{-x}\sin y + \varphi'(x)$$

由题设 $\dfrac{\partial f}{\partial x} = -f = -e^{-x}\sin y - \varphi(x)$，所以：
$$-e^{-x}\sin y + \varphi'(x) = -e^{-x}\sin y - \varphi(x)$$

整理得：
$$\varphi'(x) = -\varphi(x)$$

**第三步**：解这个最简单的微分方程。

> **秒杀技巧（必须背）**：
> - $\varphi'(x) = \varphi(x) \implies \varphi(x) = Ce^x$
> - $\varphi'(x) = -\varphi(x) \implies \varphi(x) = Ce^{-x}$（**本题**）

证明（凑微分法）：$\dfrac{d\varphi}{\varphi} = -dx \implies \ln|\varphi| = -x + C_1 \implies \varphi = \pm e^{C_1}\cdot e^{-x} = C e^{-x}$。

**第四步**：用初始条件 $f(0,0)=0$ 定常数。

$f(0,0) = e^0 \cdot \sin 0 + C\cdot e^0 = 0 + C = 0 \implies C = 0$

**最终答案**：
$$\boxed{f(x,y) = e^{-x}\sin y}$$

### 4. 综合应用：旋转体体积（课本注里给的"彩蛋"）

> **易错提醒**：这题是讲完例13.6后，老师顺手把 $f$ 表达式代入到上册学的"旋转体体积"里，**考的是跨章节综合**。

**题目**：求 $y = e^{-x}\sin x$（$x\in[0,+\infty)$）与 $x$ 轴围成的图形**绕 $x$ 轴旋转**的体积。

**套公式**（六字方针：**套公式做计算**）：
$$V = \pi\int_0^{+\infty} f^2(x)\,dx = \pi\int_0^{+\infty} e^{-2x}\sin^2 x\,dx$$

**降幂**：$\sin^2 x = \dfrac{1-\cos 2x}{2}$
$$V = \frac{\pi}{2}\int_0^{+\infty} e^{-2x}\,dx - \frac{\pi}{2}\int_0^{+\infty} e^{-2x}\cos 2x\,dx$$

**第一项**（简单）：$\displaystyle\int_0^{+\infty} e^{-2x}\,dx = \frac{1}{2}$

**第二项**（用 $\int e^{ax}\cos bx\,dx = \dfrac{e^{ax}(a\cos bx + b\sin bx)}{a^2+b^2}$）：[不定积分的积分法](1-4.%20一元函数积分学/2.%20计算/不定积分的积分法.md#^cs5xmj)

这里 $a=-2,\,b=2$，分母 $a^2+b^2=8$。原函数：
$$\int e^{-2x}\cos 2x\,dx = \frac{e^{-2x}(-2\cos 2x + 2\sin 2x)}{8} = \frac{e^{-2x}(\sin 2x - \cos 2x)}{4}$$

代入 $[0,+\infty)$：
- 上限 $x\to+\infty$：$e^{-2x}\to 0$，该项 $\to 0$
- 下限 $x=0$：$\dfrac{e^0(\sin 0 - \cos 0)}{4} = -\dfrac{1}{4}$

所以 $\displaystyle\int_0^{+\infty} e^{-2x}\cos 2x\,dx = 0 - \left(-\frac{1}{4}\right) = \frac{1}{4}$

**代回**：
$$V = \frac{\pi}{2}\cdot\frac{1}{2} - \frac{\pi}{2}\cdot\frac{1}{4} = \frac{\pi}{4} - \frac{\pi}{8} = \boxed{\frac{\pi}{8}}$$

> **避坑**：绕 $x$ 轴旋转，公式是 $\pi\int f^2\,dx$（不是 $2\pi$）。**正负都无所谓**，因为平方了。

## 三、可微概念（一维 → 二维）

> 这一节是**全讲最有"概念味"**的部分。讲课老师反复强调：**不要死记题型，要真懂"为什么"**。

### 1. 引子：矩形面积的"线性增量 + 误差"

设矩形面积 $S = xy$，给 $x$ 增量 $\Delta x$，$y$ 增量 $\Delta y$：
$$\Delta S = (x+\Delta x)(y+\Delta y) - xy = \underbrace{y\Delta x + x\Delta y}_{\text{线性主部}} + \underbrace{\Delta x \Delta y}_{\text{小尾巴}}$$

**关键观察**：
- $y\Delta x + x\Delta y$：$\Delta x$、$\Delta y$ 的**线性组合**（只含加法和数乘）
- $\Delta x \Delta y$：**乘法**（两个变量相乘），**不是线性运算**——它是**误差**

**误差有多小？** 令 $\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$（两点距离）：
$$\lim_{\rho\to 0}\frac{\Delta x\Delta y}{\rho} = 0 \quad (\text{用 } |\Delta x\Delta y|\le \tfrac{1}{2}(\Delta x^2+\Delta y^2)=\tfrac{1}{2}\rho^2 \text{ 夹逼})$$

所以 $\Delta x\Delta y$ 是**比 $\rho$ 更高阶的无穷小**，记作 $o(\rho)$。

> **口诀**：增量 = **线性主部** + **$o(\rho)$**（高阶无穷小尾巴）。

### 2. 多元可微的定义（对比一元）

| 维度 | 增量表达式 | 名称 |
|------|------------|------|
| **一元** | $\Delta y = A\Delta x + o(\Delta x)$ | $f$ 在 $x_0$ **可微** |
| **二元** | $\Delta z = A\Delta x + B\Delta y + o(\rho)$ | $f$ 在 $(x_0,y_0)$ **可微**（**全微分**） |

**精确定义**：

> 若 $\Delta z = f(x_0+\Delta x, y_0+\Delta y) - f(x_0,y_0)$ 能写成
> $$\Delta z = A\Delta x + B\Delta y + o(\rho),\quad \rho = \sqrt{(\Delta x)^2+(\Delta y)^2}$$
> 其中 $A,B$ **只与点 $(x_0,y_0)$ 有关**（与 $\Delta x,\Delta y$ 无关），则称 $f$ 在 $(x_0,y_0)$ **可微**，并称
> $$dz = A\,dx + B\,dy$$
> 为 $f$ 在该点的**全微分**。

> **避坑**：
> - 一元只有 $\Delta x$ 一个变量，所以叫"**微分**" $dy = A\,dx$。
> - 二元有 $\Delta x, \Delta y$ 两个变量，所以叫"**全微分**" $dz = A\,dx + B\,dy$（"全"= 两个增量都包含）。

### 3. 必要条件：可微 → 偏导数存在

> **若 $f$ 在 $(x_0,y_0)$ 可微，则 $\dfrac{\partial f}{\partial x}$、$\dfrac{\partial f}{\partial y}$ 在该点必存在**（**可微必可偏导**），且 $A = f_x(x_0,y_0)$，$B = f_y(x_0,y_0)$。

**证明思路**（一元对比）：
$$f_x(x_0,y_0) = \lim_{\Delta x\to 0}\frac{\Delta z}{\Delta x} = \lim_{\Delta x\to 0}\frac{A\Delta x + o(\rho)}{\Delta x} = A$$
（$\Delta y=0$ 时 $\rho=|\Delta x|$，高阶无穷小被约掉）

**所以全微分可写为**：
$$\boxed{dz = f_x\,dx + f_y\,dy}$$

### 4. 充分条件：**不**对称——偏导存在推不出可微 ⚠️

| 维度 | 一元 | 多元（重点 ⚠️） |
|------|------|--------------|
| 必要 | 可微 → 可导 | 可微 → 偏导存在 |
| 充分 | 可导 → 可微 | **偏导存在 $\nRightarrow$ 可微** |
| 关系 | **等价** | **不等价** |

> **核心考点（必背）**：
> - 一元：可微 $\iff$ 可导
> - **二元：可微 $\Rightarrow$ 偏导存在，但反过来不成立**
> - 偏导存在只保证**两个特殊方向**（平行 $x$ 轴 / $y$ 轴）的变化率存在
> - 可微要求**任意方向**的增量都满足"线性主部 + 高阶无穷小"——条件强得多

> **数一专属思考题**（数二三不考）：所有方向的方向导数都存在，能不能推出可微？
> **答：也不能**。方向导数描述"变化率"，可微描述"增量结构"，**是两个不同的概念**。

### 5. 几何意义（强烈建议画图理解）

**一元情形**：
- 曲线在 $x_0$ 点有**切线**（$A = f'(x_0)$ 是斜率）
- $\Delta y$（曲线真实增量）= $dy$（切线增量）$+$ $o(\Delta x)$（误差）
- **可微 = 切线上的点可以替代曲线上的点，误差是高阶无穷小**

**二元情形**：
- 曲面在 $(x_0,y_0)$ 点有**切平面**（不是切线！）
- $\Delta z$（曲面上点的真实差）= 切平面上点的差 $+$ $o(\rho)$
- **可微 = 切平面上的点可以替代曲面上的点，误差是高阶无穷小**

> **避坑**：多元可微 → **切平面**，不是切线。一元的结论是"切线"，多元是"切平面"，**维度都升了一级**。

> **口诀总结**：
> - 一元可微 = **切线**能替代曲线，误差 $o(\Delta x)$
> - 二元可微 = **切平面**能替代曲面，误差 $o(\rho)$
> - 切线斜率 = 导数；切平面方程系数 = 偏导数

## 四、本讲总结：方法对照表（续）

|| 一元函数 | 多元函数 | 关系 |
|----------|----------|--------|
| 变限积分求导 | 变限积分求偏导 | ✅ 方法完全一样，**对 $x$ 偏导时 $y$ 当常数** |
| 已知 $F'$ 求 $F$（加 $C$）| 已知 $f_y$ 求 $f$（加 $\varphi(x)$）| ⚠️ 加的是"**反方向**"的任意函数 |
| 微分 $dy = A\,dx$ | 全微分 $dz = A\,dx + B\,dy$ | ✅ 形式一致，多元多了 $B\,dy$ 项 |
| 可微 $\iff$ 可导 | 可微 $\Rightarrow$ 偏导存在（**反之不成立**）| ❌ 多元**不等价**——核心考点 |

---

> **一句话记住这一讲**：
> - 变限积分求偏导：能拆就拆，拆不开就**换元**，**先代值再求导**是秒杀；
> - 反问题：对哪个变量积分，就加**对面那个变量的任意函数**；
> - 可微：增量 = 线性主部 + $o(\rho)$，多元里**可微必可偏导，反之不成立**——这是和一元的**根本区别**。
