## 一、微分方程的概念

### 1. 什么是微分方程

「考点」**微分方程的两大必备条件**:

- ① 一个**方程**(等式)
- ② 含**未知函数的导数**(或微分)

教材原话:

> 表示未知函数及其导数(或者微分)与自变量之间关系的方程称为微分方程,一般写成
> $$F[x, y, y', \dots, y^{(n)}] = 0 \quad\text{或}\quad y^{(n)} = f[x, y, y', \dots, y^{(n-1)}].$$

研究对象是 $y = y(x)$. 我们的目标就是把它求出来。

「易错提醒」**研究对象 ≠ 方程形式**. 即使显不出 $y$ 或 $x$ 或 $y'$, 只要含有一项导数,就符合定义,就是微分方程。

### 2. 微分方程的阶

「定义」微分方程中**未知函数的最高阶导数的阶数**称为微分方程的**阶**.

教材例:

$$y''' - y'' + 6y = 0 \quad \Longleftarrow\quad \text{三阶微分方程(除 }y'''\text{ 外,其他均可缺)}$$

### 3. 常微分方程 vs 偏微分方程

「分类」按自变量个数:

| 类别             | 含义                       | 大纲要求                  |
| -------------- | ------------------------ | --------------------- |
| **常微分方程(ODE)** | 未知函数是**一元**函数 $y = y(x)$ | ✅ 考研必考                |
| **偏微分方程(PDE)** | 含**未知函数的偏导数**            | ⚠️ 大纲不要求,少量题目涉及(强化阶段) |

常微分方程实例:

$$y''' - y'' + 6y = 0, \qquad y\,\mathrm{d}x - (x + \sqrt{x^2 + y^2})\,\mathrm{d}y = 0$$

### 4. 线性微分方程

「定义」**$n$ 阶线性微分方程**的标准形:

$$a_n(x)\, y^{(n)} + a_{n-1}(x)\, y^{(n-1)} + \cdots + a_1(x)\, y' + a_0(x)\, y = f(x)$$

要求 $a_n(x) \ne 0$($a_n$ 不能为 0,否则不叫 $n$ 阶).

「考点分类」(4 类怎么叫,4 种组合):

| 分类维度 | 两种叫法 | 触发条件 |
|---------|---------|---------|
| **按系数** | **常系数** vs **变系数** | $a_k(x)$ 都是常数 vs 至少有一个是 $x$ 的函数 |
| **按右端** | **齐次** vs **非齐次** | $f(x) \equiv 0$ vs $f(x) \not\equiv 0$ |

4 种组合任意拼(讲次名都不重复):

- $n$ 阶线性微分方程(母概念)
- $n$ 阶常系数线性微分方程
- $n$ 阶齐次线性微分方程
- $n$ 阶常系数非齐次线性微分方程

「避坑」**变系数 ≠ 不确定**. $a_k(x)$ 是 $x$ 的**已知**函数(只是形式上含 $x$),不是未知的. 这里"未知的研究对象"只有 $y$ 一个,**其他都不是未知**.

### 5. 微分方程的解

「定义」把函数代到方程里,等式两边恒等,这个函数就叫微分方程的**解**.
  
「几何」微分方程解的图形称为**积分曲线**.
> "积分曲线"是个独立名词,看到它要立即反应 = "解的几何表达".

### 6. 微分方程的通解(独立常数)

「定义」若微分方程的解中含有的**独立常数的个数**等于微分方程的**阶数**,则该解称为微分方程的**通解**.

「秒杀技巧」**"独立" ≠ "任意"**:

- **独立**:经任何恒等变形都不能使常数个数减少. 即常数前的两个函数必须**线性无关**($\frac{y_1(x)}{y_2(x)} \ne \text{常数}$).
- **任意**:能取尽某范围内任意值(可见通解里的常数并**不**一定非要"任意").

「正反例」:

| 表达式 | 是二阶通解吗? | 原因 |
|--------|:---:|------|
| $y = C_1\sin x + C_2 \cdot 2\sin x = C\sin x$ | ❌ | $\frac{\sin x}{2\sin x} = \frac{1}{2}$ 是常数,两个函数**线性相关**,常数可合并,只剩 1 个独立常数 |
| $y = \mathrm{e}^x(C_1 + C_2 x)$ | ✅ | $\frac{\mathrm{e}^x}{\mathrm{e}^x \cdot x} = \frac{1}{x}$ 不是常数,两函数**线性无关**,$C_1, C_2$ 是 2 个独立常数 |

「避坑」一些教材写"任意常数",严谨不如"独立常数". 例 15.4 会遇到常数取值受限制的情形(本题不在本节范围).

### 7. 初始条件与特解

「定义」**初始条件** = 确定通解中常数的条件.

标准写法:

$$y(x_0) = a_0,\quad y'(x_0) = a_1,\quad \cdots,\quad y^{(n-1)}(x_0) = a_{n-1}$$

> $n$ 阶方程 → 必须给 **$n$ 个初始条件** → 才能定下来所有常数 → 得到**特解**.

「定义」给出初始条件求特解的问题,数学上叫**定解问题**(也叫科西问题).

「秒杀技巧」**特解唯一性 ↔ 定解问题**:
- 通解 = 一族积分曲线(无穷多条)
- 每一条初始条件 = 一条曲线上"过哪一点" → 砍掉其他,只留唯一一条 → 唯一特解

## 二、概念总结:微分方程研究的两件事

老王的一句总结 —— 对于微分方程,我们**只关心两件事**:

| # | 问题 | 答案 |
|---|------|------|
| ① | **有没有解**(解的存在性) | 微分方程理论保证:合适条件下一定有解. 考研不考,研究生阶段再深究 |
| ② | **有没有唯一解**(解的唯一性) | 给定初始条件 → 唯一解(特解) |

> 抠这两个问题,就已经抓住了微分方程的全部研究主题. 考研范围内的题目,都默认"有解 + 给条件定特解"这条主轴.

## 三、例题精讲

### 例 15.1(概念辨析 · 第二充分条件)

> 设 $y = f(x)$ 是方程 $\boxed{y'' - 2y' + 4y = 0}$ 的一个解,若 $f(x_0) > 0$,且 $f'(x_0) = 0$,则 $f(x)$ 在点 $x_0$ 处(  ).
> (A) 取得极大值  (B) 取得极小值  (C) 某个邻域内单调增加  (D) 某个邻域内单调减少

「思路 · 避坑」**不要去解方程**.

- ① 方程只有一个解的**信息**用上了:把 $x_0$ 代入得
$$f''(x_0) - 2f'(x_0) + 4f(x_0) = 0$$
- ② 加上题给条件 $f(x_0) > 0$, $f'(x_0) = 0$:
$$f''(x_0) = -4 f(x_0) < 0$$
- ③ **判别极值的第二充分条件**:$f'(x_0) = 0$ 且 $f''(x_0) < 0$ → $x_0$ 是**极大值点**.

「答案」(A)

「秒杀技巧总结」**看到微分方程 + 极值判断 → 别求解,直接代入点用充分条件判**.

「易错提醒」**微分方程任意阶可导** (教材导出页面自 305-478-3.md:115 注):
- 这里 $f''$ 出现 → 可以再求导 → $f'''$ 也存在 → ... → 任意阶可导
- 结论:只要题中出现 $f$ 满足某 $n$ 阶微分方程,那 $f^{(n)}$ 是连续的,后面所有更高阶导数也连续.

### 例 15.2(综合 · 极限 + 洛必达)

> 设 $y = y(x)$ 是二阶常系数线性微分方程 $y'' + py' + qy = \mathrm{e}^{3x}$ 满足初始条件 $y(0) = y'(0) = 0$ 的特解,则
> $$\lim_{x \to 0} \frac{\ln(1 + x^2)}{y(x)} = \text{\_\_\_\_\_}.$$
> (教材导出页面自 305-478-3.md:121-132)

「步骤拆解」

**第 1 步:从方程里"挖"出所有能用的导数值.**

把 $x = 0$ 代入 $y'' + py' + qy = \mathrm{e}^{3x}$,得

$$y''(0) + p\, y'(0) + q\, y(0) = \mathrm{e}^{0} = 1$$

由 $y(0) = y'(0) = 0$,推出 $y''(0) = 1$.

**第 2 步:把原极限替换 + 判型.**

分子 $\ln(1+x^2) \sim x^2$ 当 $x\to 0$,故原极限等于

$$\lim_{x\to 0} \frac{x^2}{y(x)}$$

这是 $\frac{0}{0}$ 型(分子 $\to 0$,分母 $\to y(0) = 0$).

**第 3 步:连续用两次洛必达.**

$$\lim_{x\to 0} \frac{2x}{y'(x)} \quad(\text{仍 }\tfrac{0}{0}) \;\longrightarrow\; \lim_{x\to 0} \frac{2}{y''(x)} = \frac{2}{y''(0)} = \frac{2}{1} = 2$$

「答案」$\boxed{2}$

「秒杀技巧总结」这类题**不能也不需要**解微分方程 —— 思路是**反向利用**方程:已知函数满足的等式 = **函数与其各阶导数之间的关系**. 把题目信息(各阶导数值)挖出来,代入极限题直接算.

## 四、本节方法总览(速查表)

| 题型/任务 | 该用哪个概念? | 该写什么 |
|----------|--------------|---------|
| 看到"未知函数含导数的等式" | **微分方程定义** | 抓"等式 + 含导数"两个条件 |
| 问方程是几阶 | **阶的定义** | 找不能缺的最高阶导数项 |
| $a_k(x)$ 含 $x$ 表达式 | **线性,但变系数** | |
| 右端 $f(x) \equiv 0$ | **齐次** | |
| 给 $y = e^x(C_1 + C_2 x)$ 问通解 | **独立常数 = 阶数** | 验证 $\frac{e^x}{x e^x} = \frac{1}{x} \ne \text{常数}$ |
| $n$ 阶微分方程 + 给定积分值 | **特解 / 定解问题** | 必须有 $n$ 个初始条件 |
| 微分方程 + 极值判别 | **直接代入点 + 第二充分条件** | 不解方程 |
| 微分方程 + 极限 (0/0 型) | **反向挖各阶导数值** + 洛必达 | 不解方程 |

## 五、易错提醒(踩坑汇总)

> [!warning] **核心坑 1:把"通解 = 有任意常数的解"**
> 错. 必须"**独立**"常数. $C_1 \sin x + C_2 \cdot 2 \sin x$ 看似两个常数,实则 **线性相关**, 系数可合并, 不是通解.

> [!warning] **核心坑 2:通解里常数一定是"任意常数"**
> 不一定(教材导出页面自 305-478-3.md:96 明确说"常数并非一定是任意常数,可能是在一定范围内取值的常数, 如例 15.4"). 严谨说法 = **独立常数**, 不是任意常数.

> [!warning] **核心坑 3:微分方程缺项判断**
> $y, y', x$ **都可以缺**, **仅有**"$y$ 的某阶导数"这一项**不能缺**. 哪怕方程里就一个 $y^{(n)} = 0$, 它也是 $n$ 阶微分方程.

> [!warning] **核心坑 4:研究对象只有 $y$ 一个**
> 凡有 $a_k(x)$ 跟 $x$ 有关, $a_k(x)$ 是 **已知** 的(只是形式上含 $x$), **不是未知**. 别看到 $x$ 就以为是变量未知.
